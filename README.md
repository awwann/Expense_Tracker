# Expense Tracker - Трекер расходов

## Информация об авторе
**Автор:** Сумская Анна
**Дата создания:** Май 2026

## Описание программы
Простая программа для учёта денег. Помогает следить за тем, куда уходят карманные расходы. Написана на Python с удобным календарём.

## Возможности программы:
- Учет карманных расходов
- Планирование бюджета на развлечения
- Анализ трат
- Контроль расходов за определенный период
- Сохранение истории трат

## Установка и запуск

### Требования:
- Стандартная библиотека Python (tkinter, json)
- Python 3.6 или выше

### Запуск программы
1. Открой папку с программой
2. Открой терминал в этой папке
3. Запусти код

## Используемые технологии
- **Python 3** - Язык программирования
- **Tkinter** - Стандартная GUI-библиотека Python
- **JSON** - Формат хранения данных
- **Git** - Система контроля версий
